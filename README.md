# 🎮 Advanced Number Guessing Game (CLI)

A powerful and interactive command-line based Number Guessing Game built using Python.
This project demonstrates strong problem-solving, logic building, and real-world development practices.

---

# 🚀 Features

* 🎯 Multiple Difficulty Levels

  * Easy (1–50)
  * Medium (1–100)
  * Hard (1–200)

* 🧠 Smart AI Hint System

  * "Way too high" / "Slightly low" hints

* 🔢 Attempt Counter

  * Tracks number of guesses

* 🏆 Leaderboard System

  * Stores player scores using JSON

* 📊 Analytics Graph

  * Visualizes attempts history using Matplotlib

* 🔁 Replay Option

* ❌ Exit anytime (`exit` command)

---

# 🛠 Tech Stack

* Python 3
* Modules:

  * random
  * json
  * matplotlib

---

# ▶️ How to Run

```bash
pip3 install matplotlib
python3 guess_game.py
```
